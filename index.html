角の折り返し

「background」

{background バックグラウンド : -color -image -repeat -position -attachment -clip -size -origin;}
background-sizeプロパティの値は、backgroundpositionプロパティの値にスラッシュ(/)で続けて指定

例

background: linear-gradient(to left bottom,transparent 50%,rgba(0,0,0,.4) 0) no-repeat 100% 0 / 2em 2em;

背景画像のスタート位置を幅100%、高さ0%と指定し、幅と高さをそれぞれ2emとする。

折り返し28 * 28 とした場合、背景の切り落としは 28 / √2 となる。

解説

下記三角関数を元に計算

高さ:1、幅:√3、斜辺:2で角が30、60、90の直角三角形

cos30 = √3 / 2
sin30 = 1 / 2

折り返しを30度として、サイズを28pxとした場合。右上角から折り返しまでの垂線が28pxとなる。

折り返しの影の幅をx、高さをyとする。ここでこの三角形のcosとsinを求めると、
cos30= 28 / y
sin30= 28 / x
となる。先ほどの基本式と合わせると、 28 / y= √3 / 2
28 / x = 1 / 2
となり、xとyの値が求まる。この場合は、y = 56 / √3　、x = 56 となる。
